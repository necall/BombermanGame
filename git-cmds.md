## *First, Clone the project in midterm branch*



```bash
------create branch for your work------
// create a new branch:
git checkout -b [YOUR_BRANCH_NAME]

------after finishing your work--------
// or switch to your branch:
git checkout [YOUR_BRANCH_NAME]

// add changed files
git add [FILE_NAME]

// commit the request
git commit -m 'MESSAGE'

// switch to midterm branch
git checkout midterm

// pull!!!
git pull origin midterm

// switch back to your branch:
git checkout [YOUR_BRANCH_NAME]

// merge
git merge midterm

// if conflicts happen, fix it first!!!

// push your request
git push origin [YOUR_BRANCH_NAME]

// then go to github to pull a request to master3

------Others:-------------------------
// delete your branch:
git branch -D [YOUR_BRANCH_NAME]

```
